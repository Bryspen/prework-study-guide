Prework Study Guide

HTML

  • Stands for Hyper Text Markup Language.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Pictures can be added using the "img src" tag and links can be added using the "a href" tag.

CSS

  • CSS, or Cascading Style Sheets is a programming language used by developers to define how webpage content is presented to users.
  • >A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • We can use a comma between selectors to avoid Do Not Repeat Yourself(DRY) code.

Git

  • Git bash is a program that allows us to access the command line in windows.
  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • code . allows us to access current directory in visual studio.
